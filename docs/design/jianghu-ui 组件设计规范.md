## 📌 一、全局基础规范（所有组件统一遵循）
| 项         | 具体规范                     | 参考值/示例                     |
|------------|------------------------------|---------------------------------|
| **圆角**   | 基础元素（按钮/输入框）：4px<br>大容器（卡片/弹窗）：6–8px | 按钮统一 4px，卡片统一 6px      |
| **边框**   | 统一 1px 细边框，默认浅灰色  | 边框色值：`#E5E7EB`<br>hover 时加深为 `#D1D5DB` |
| **色彩系统** | 主色：低饱和度功能色（如图中绿色）<br>中性色：浅灰/白/深灰构建层级<br>辅助色：红（删除）/绿（编辑）区分操作 | 主色：`#10B981`<br>文字主色：`#1F2937`<br>背景色：`#FFFFFF`/`#F9FAFB` |
| **字体**   | 无衬线字体（Inter/Roboto），层级清晰 | 标题：16–18px / 字重 600<br>内容：14px / 字重 400<br>辅助文字：12px / 字重 400 |
| **间距**   | 模块间 24px，组件间 16px，元素间 8px | 保证界面「呼吸感」，避免拥挤 |

## 🛠️ 二、核心组件详细规范
### 1. 按钮组件
- **视觉规范**：
  - 圆角 4px，高度 32–36px，文字 14px / 字重 500
  - 主按钮（新增/确认）：主色填充 + 白色文字
  - 次要按钮：白色填充 + 主色边框 + 主色文字
  - 危险按钮（删除）：白色填充 + 红色边框 + 红色文字
- **交互规范**：
  - hover：主按钮颜色加深（`#059669`），次要按钮背景变浅（`#F3F4F6`），危险按钮背景变红（`#EF4444`）+ 文字变白
  - 点击：轻微缩小或颜色加深反馈

### 2. 输入框/筛选器组件
- **视觉规范**：
  - 圆角 4px，边框 1px `#E5E7EB`，背景白色
  - 占位符文字：`#9CA3AF`（浅灰），输入文字：`#1F2937`（深灰）
- **交互规范**：
  - hover：边框变为 `#D1D5DB`
  - 聚焦：边框变为主色 `#10B981`，添加极淡内阴影
  - 禁用：背景 `#F3F4F6` + 文字 `#9CA3AF`

### 3. 表格组件
- **视觉规范**：
  - 单元格圆角 4px，分割线 1px `#E5E7EB`
  - 表头：背景 `#F9FAFB` + 文字 14px / 字重 500
  - 内容行：背景白色，hover 时背景变为 `#F3F4F6`
  - 操作按钮（编辑/删除）：图标+文字组合，绿色/红色，无背景
- **交互规范**：
  - 行 hover：背景高亮 + 轻微阴影
  - 操作按钮 hover：颜色加深 + 浅灰背景

### 4. 卡片组件
- **视觉规范**：
  - 圆角 6–8px，边框 1px `#E5E7EB`，背景白色
  - 内边距 16px，标题 16px / 字重 600，内容 14px
  - 阴影：极淡外阴影（`0 1px 3px 0 rgba(0,0,0,0.1)`）
- **交互规范**：
  - hover：阴影加深（`0 2px 4px 0 rgba(0,0,0,0.1)`）+ 边框变为 `#D1D5DB`

### 5. 图标组件
- **视觉规范**：
  - 线性图标为主，线条粗细 1.5px
  - 功能图标（编辑/删除）：使用对应功能色（绿/红）
  - 通用图标：颜色与文字统一（`#1F2937`），大小 16–18px
- **交互规范**：
  - hover：颜色加深 + 浅灰背景
